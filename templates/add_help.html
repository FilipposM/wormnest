The Following parameters are needed (either GET or POST):
  • path: The Path to the file to be served with minified URL.
  • clicks: The number of clicks that will expire the URL. If blank the URL will not expire.
  • alias: The alias to be used. If blank a random string will be used.
  • filename: The filename that will be served from the minified URL. If blank a default filename will be used.
  • unchecked: If "True" the "path" is not checked for existence. This can lead to 404 errors of the minified URL. Defaults to "False".
